#### Using Corner Radii with SASS Border Mixin

The mixin contains a line for radius, and is passed as a parameter. By default the parameter is $corner-200, or 8px.
```

@mixin border($width: $border-width-100, $style: solid, $color:$color-border-primary-brand, $radius:$corner-200) {
  border: $width $style $color;
  border-radius: $radius
}

```

To change the border-radius add one of the corner-radius tokens. For example:

```
.border-primary-brand-corner-none {
  @include border($radius:$corner-none);
}
```
